> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scale the Elasticsearch HotpotQA baseline to 100k documents and compare improved iterative multi-hop retrieval variants against hybrid search for the Saturday report.

**Architecture:** Keep the current Elasticsearch-only baseline. Build a separate 100k index, add small iterative query expansion variants inside the existing retriever, extend benchmark method mapping, then run controlled comparisons with quality and latency metrics. Avoid MDR and IRCoT implementation in this sprint because they require new training data, LLM traces, or model infrastructure.

**Tech Stack:** Python, `ir_datasets`, Elasticsearch 8.x, `sentence-transformers`, `BAAI/bge-small-en-v1.5`, pytest, existing benchmark scripts.

This sprint produces report-ready evidence for:

```python
    def _expand_query(self, query: str, hit: dict[str, Any], context_chars: int, expansion_mode: str = "context") -> str:
        title = str(hit.get("title", "") or "")
        text = str(hit.get("text", "") or "").replace("\n", " ")
        if expansion_mode == "title":
            parts = [query, title]
        elif expansion_mode == "sentence":
            parts = [query, title, self._select_sentence(query, text)]
        elif expansion_mode == "context":
            parts = [query, title, text[:context_chars]]
        else:
            raise ValueError(f"Unknown expansion mode: {expansion_mode}")
        return " ".join(part for part in parts if part)

    def _select_sentence(self, query: str, text: str) -> str:
        sentences = [sentence.strip().rstrip(".") for sentence in SENTENCE_RE.split(text) if sentence.strip()]
        if not sentences:
            return ""
        query_terms = {token.lower() for token in TOKEN_RE.findall(query)}
        return max(sentences, key=lambda sentence: self._sentence_overlap_score(query_terms, sentence))

    def _sentence_overlap_score(self, query_terms: set[str], sentence: str) -> tuple[int, int]:
        sentence_terms = {token.lower() for token in TOKEN_RE.findall(sentence)}
        return (len(query_terms & sentence_terms), len(sentence_terms))
```
