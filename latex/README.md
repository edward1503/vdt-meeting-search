# Mentor Report LaTeX Draft

This folder contains a mentor-facing LaTeX report draft for the retrieval system.

## Build

The source is written for XeLaTeX because the report uses Vietnamese text.

```powershell
cd latex
xelatex mentor-report.tex
xelatex mentor-report.tex
```

If XeLaTeX is not installed, install a TeX distribution such as TeX Live or MiKTeX.

## Files

- `mentor-report.tex`: main report source.
